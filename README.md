## 🚀 Features

### Core Functionality
- 🎯 Smart Command Generation: Build Nmap commands visually with real-time preview  
- ⚡ Live Updates: Command updates instantly as you select options  
- 📋 One-Click Copy: Copy generated commands to clipboard with confirmation  
- 🎨 Professional UI: Dark-themed cyber interface with smooth animations  

### Advanced Features
- 🧠 Smart Presets:
  - Quick Scan  
  - Stealth Recon  
  - Full Aggressive Scan  
  - Bug Bounty Scan  

- 📚 Educational Panel: Learn what each flag does and why it's used  
- 📖 Built-in Cheat Sheet: Comprehensive Nmap reference guide  
- 📊 Sample Output: See what the command would produce  
- 💾 Command History: Automatically saves your last 10 commands  
- 📤 Export Options: Export configurations in multiple formats  
- 🔗 Share Links: Generate shareable configurations (UI ready)  

### UI/UX Features
- 🌙 Dark Mode: Cyber-themed interface optimized for long sessions  
- 📱 Mobile Responsive: Works perfectly on all devices  
- ⌨️ Keyboard Shortcuts: ESC to close modals  
- 🎭 Smooth Animations: Professional transitions and micro-interactions  
- 🔔 Toast Notifications: Non-intrusive feedback system  

---

## 🛠️ Tech Stack

- Frontend: HTML5, CSS3, JavaScript (ES6+)  
- Styling: Tailwind CSS (via CDN)  
- Icons: Font Awesome 6  
- Fonts: Inter & JetBrains Mono  
- Storage: LocalStorage  
- No Build Process: Pure vanilla web technologies
