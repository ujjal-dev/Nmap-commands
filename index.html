Nmap Cheat Sheet

Scan Techniques

  • -sS - TCP SYN scan
  • -sT - TCP connect scan
  • -sU - UDP scan
  • -sA - TCP ACK scan

Port Specification

  • -p 80 - Single port
  • -p 80,443 - Multiple ports
  • -p 20-25 - Port range
  • -p- - All ports

Detection Options

  • -sV - Version detection
  • -O - OS detection
  • -A - Aggressive scan
  • --traceroute - Traceroute

Timing Templates

  • -T0 - Paranoid (very slow)
  • -T3 - Normal (default)
  • -T4 - Aggressive (faster)
  • -T5 - Insane (very fast)

Output Formats

  • -oN - Normal output
  • -oX - XML output
  • -oG - Greppable output
  • -oA - All formats

Script Scanning

  • -sC - Default scripts
  • --script - Custom scripts
  • --script-args - Script args
  • --script-help - Script help
